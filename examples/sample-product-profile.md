# 产品理解报告

## 产品定位
PM Agent 是一个运行在 Cursor IDE 中的 AI 产品经理协作搭档，帮助产品经理从需求对齐到 PRD 输出全流程自动化。

## 核心功能
1. 工作流自动调度：从需求对齐、场景推演、详细设计到 PRD 输出，AI 帮你按标准流程走完全程，不用再担心遗漏环节
2. 智能需求文档生成：告诉 AI 你要做什么，它帮你生成完整的需求文档，包含功能列表、用户场景、验收标准，省掉 80% 的文档撰写时间
3. 自动评审与自检：写完文档自动帮你检查遗漏和矛盾，在评审会前就发现问题
4. UI 原型生成：从需求描述直接生成页面原型，不用等设计师排期

## 目标受众
- **主要受众**：产品经理 — 每天大量时间花在写需求文档上，希望提效
- **次要受众**：独立开发者 — 自己兼任 PM，但不擅长写规范的需求文档

## 差异化卖点
1. 全流程覆盖：不是只帮你写某一段，而是从需求对齐到 PRD 定稿全程陪跑，市面上的 AI 写作工具做不到这种深度
2. 嵌入开发工具：直接在 Cursor IDE 里用，不用切换工具，和代码开发无缝衔接
3. 结构化输出：生成的不是散文，而是标准化的需求文档，包含状态机、验收标准、异常处理等专业内容

## 可推广亮点（小红书视角）
1. "3 分钟出 PRD"：时间效率的对比最容易引起共鸣，适合做对比类内容
2. AI 做产品经理的概念新颖：AI 替代/辅助特定职业的话题在小红书上有热度
3. 可视化的前后对比：展示 AI 生成的需求文档 vs 手写文档的效率差异
4. 独立开发者故事：独立开发者用 AI 工具提效的真实经历容易引发共鸣
